💡 说明：OpenClaw官方推荐在Linux或macOS上运行，但在Windows上可以通过WSL（Windows Subsystem for Linux）完美运行。本文将详细介绍每一步操作。

一、准备工作

1检查系统要求

  • Windows 10 版本 2004 或更高版本（Build 19041+）
  • Windows 11（推荐）
  • 至少4GB内存（建议8GB+）
  • 至少10GB可用磁盘空间

2启用虚拟化

打开任务管理器 → 性能 → CPU，确认"虚拟化"已启用。如果没有启用，需要在BIOS中开启。

二、安装WSL（Windows子系统 for Linux）

1以管理员身份打开PowerShell

右键点击开始菜单 → 选择"Windows PowerShell (管理员)"

2安装WSL2

在PowerShell中执行以下命令：

PS C:\Windows\system32> wsl --install
⚠️ 注意：如果提示"需要启用虚拟机平台功能"，执行以下命令：
PS C:\Windows\system32> dism.exe /online /enable-feature /featurename:Microsoft-Windows-Subsystem-Linux /all /norestart

PS C:\Windows\system32> dism.exe /online /enable-feature /featurename:VirtualMachinePlatform /all /norestart

然后重启电脑！

3设置WSL2为默认版本

PS C:\Windows\system32> wsl --set-default-version 2

4安装Ubuntu

PS C:\Windows\system32> wsl --install -d Ubuntu

安装完成后会提示设置用户名和密码：

Enter new UNIX username: pili4
New password: ********
Retype password: ********
✅ 提示：设置密码时不会显示任何字符，这是正常的，输入后回车即可。

5打开Ubuntu终端

在开始菜单找到"Ubuntu"并打开，或者在PowerShell中执行：

PS C:\Windows\system32> wsl -d Ubuntu

6更新Ubuntu软件源

pili4@DESKTOP:~$ sudo apt update && sudo apt upgrade -y
⏳ 提示：第一次执行`sudo`需要输入密码，输入时不会显示任何字符。

三、安装Docker Desktop

1下载Docker Desktop

访问 Docker 官网下载：https://www.docker.com/products/docker-desktop/

下载 Windows 版本（Docker Desktop Installer.exe）

2安装Docker Desktop

双击安装程序，勾选"Use WSL 2 instead of Hyper-V"，然后点击"Install"

⚠️ 注意：安装过程中会短暂中断网络连接，这是正常的。

3启动Docker Desktop

安装完成后，在开始菜单找到"Docker Desktop"并启动。首次启动需要等待几分钟初始化。

✅ 验证：在Ubuntu终端执行以下命令验证Docker是否正常运行：
pili4@DESKTOP:~$ docker --version
# 输出：Docker version 24.0.x, build xxxxx

pili4@DESKTOP:~$ docker ps
# 输出：CONTAINER ID IMAGE COMMAND CREATED STATUS PORTS NAMES

四、安装OpenClaw

1在Ubuntu中安装Docker Compose

pili4@DESKTOP:~$ sudo apt install docker.io -y

pili4@DESKTOP:~$ sudo systemctl enable docker

pili4@DESKTOP:~$ sudo usermod -aG docker $USER
⚠️ 注意：执行完`usermod`后需要重新打开终端或重新登录WSL使权限生效。

2克隆OpenClaw仓库

pili4@DESKTOP:~$ cd ~ && git clone https://github.com/openclaw/openclaw.git

3进入目录并查看配置

pili4@DESKTOP:~$ cd openclaw && ls -la

会看到以下文件：

  • docker-compose.yml - Docker编排文件
  • .env.example - 环境变量示例
  • README.md - 说明文档

4配置环境变量

pili4@DESKTOP:~/openclaw$ cp .env.example .env
✏️ 重要：编辑`.env`文件，填入你的API密钥：
pili4@DESKTOP:~/openclaw$ nano .env

修改以下内容：

# OpenAI API配置
OPENAI_API_KEY=sk-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx

# 或者使用 Anthropic Claude
ANTHROPIC_API_KEY=sk-ant-xxxxxxxxxxxxxxxxxxxxxxxx

# Telegram配置（可选）
TELEGRAM_BOT_TOKEN=your_telegram_bot_token
# 保存并退出：按 Ctrl+O，然后 Ctrl+X

5启动OpenClaw

pili4@DESKTOP:~/openclaw$ docker-compose up -d
✅ 启动成功！

6查看运行状态

pili4@DESKTOP:~/openclaw$ docker ps

应该看到以下容器运行中：

  • openclaw-gateway - 网关服务
  • openclaw-agent - Agent服务
  • openclaw-web - Web界面

五、配置与启动

1访问Web界面

打开浏览器访问：

🔗 http://localhost:18789

应该能看到OpenClaw的Web界面了！

2配置Telegram（可选）

如果你想通过Telegram与OpenClaw交互：

  1. 在Telegram中搜索@BotFather
  2. 发送命令/newbot创建新机器人
  3. 按照提示设置名称和用户名
  4. 复制Bot Token并填入.env文件
  5. 重启服务：docker-compose restart

3常用命令

# 查看日志
pili4@DESKTOP:~/openclaw$ docker-compose logs -f

# 重启服务
pili4@DESKTOP:~/openclaw$ docker-compose restart

# 停止服务
pili4@DESKTOP:~/openclaw$ docker-compose down

# 更新OpenClaw
pili4@DESKTOP:~/openclaw$ git pull && docker-compose up -d

六、常见问题

Q1: Docker启动失败怎么办？

检查WSL2是否正确安装：

PS C:\Windows\system32> wsl --list --verbose

Q2: 拉取Docker镜像很慢？

建议配置Docker镜像加速器（国内源）：

pili4@DESKTOP:~$ sudo mkdir -p /etc/docker
pili4@DESKTOP:~$ sudo tee /etc/docker/daemon.json <
pili4@DESKTOP:~$ sudo systemctl daemon-reload
pili4@DESKTOP:~$ sudo systemctl restart docker

Q3: 如何查看OpenClaw日志？

pili4@DESKTOP:~/openclaw$ docker-compose logs -f agent

Q4: WSL2内存占用过高？

创建~/.wslconfig文件限制内存：

pili4@DESKTOP:~$ notepad.exe ~/.wslconfig 
[wsl2]
memory=4GB
processors=2

然后在PowerShell中执行：

PS C:\Windows\system32> wsl --shutdown
