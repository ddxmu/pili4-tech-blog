OpenClaw Windows WSL2 安装教程
📢 说明：本文根据OpenClaw官方文档整理，官方推荐使用npm/pnpm方式安装，而不是Docker（Docker是可选的）。Windows用户需要通过WSL2运行。

一、前置要求

✅系统要求

  • Windows 10 版本 2004 或更高（Build 19041+）
  • Windows 11（推荐）
  • Node.js ≥ 22（官方要求）
  • 至少4GB内存（建议8GB+）
  • 至少5GB可用磁盘空间

二、安装WSL2（Windows子系统 for Linux）

1以管理员身份打开PowerShell

右键点击开始菜单 → 选择"Windows PowerShell (管理员)"

2安装WSL2

在PowerShell中执行：

PS C:\Windows\system32> wsl --install
⚠️ 注意：如果提示需要启用虚拟机平台功能，执行以下命令后重启电脑：
PS C:\Windows\system32> dism.exe /online /enable-feature /featurename:Microsoft-Windows-Subsystem-Linux /all /norestart

PS C:\Windows\system32> dism.exe /online /enable-feature /featurename:VirtualMachinePlatform /all /norestart

3设置WSL2为默认版本

PS C:\Windows\system32> wsl --set-default-version 2

4安装Ubuntu

PS C:\Windows\system32> wsl --install -d Ubuntu

安装完成后设置用户名和密码（密码输入时不显示是正常的）：

Enter new UNIX username: pili4
New password: ********
Retype password: ********

5打开Ubuntu终端

在开始菜单找到"Ubuntu"并打开

6更新软件源

pili4@DESKTOP:~$ sudo apt update && sudo apt upgrade -y

三、安装Node.js

1使用fnm安装Node.js（推荐）

pili4@DESKTOP:~$ curl -fsSL https://fnm.vercel.app/install | bash
pili4@DESKTOP:~$ source ~/.bashrc
pili4@DESKTOP:~$ fnm install 22 && fnm use 22

2验证Node.js安装

pili4@DESKTOP:~$ node --version
# 输出：v22.x.x

pili4@DESKTOP:~$ npm --version
# 输出：10.x.x

四、安装OpenClaw

1全局安装OpenClaw

💡 提示：官方推荐使用npm或pnpm安装。使用pnpm可以获得更好的开发体验。

方式A：使用npm（最简单）

pili4@DESKTOP:~$ npm install -g openclaw@latest

方式B：使用pnpm（推荐）

pili4@DESKTOP:~$ npm install -g pnpm
pili4@DESKTOP:~$ pnpm add -g openclaw@latest

2验证安装

pili4@DESKTOP:~$ openclaw --version

五、配置与运行

1运行初始化向导（推荐）

🎯 官方推荐：使用向导可以一步步完成所有配置，包括API密钥、频道连接、技能安装等。
pili4@DESKTOP:~$ openclaw onboard --install-daemon

向导会提示你：

  1. 配置API密钥（OpenAI/Anthropic等）
  2. 设置Telegram/WhatsApp等频道
  3. 安装技能（可选）
  4. 安装后台服务（开机自启）

2手动启动Gateway（可选）

如果不想用向导，可以手动启动：

pili4@DESKTOP:~$ openclaw gateway --port 18789 --verbose

3访问控制台

打开浏览器访问：

🔗 http://127.0.0.1:18789/

在设置中填入Gateway Token即可使用。

六、常用命令

日常使用命令

# 启动Gateway
pili4@DESKTOP:~$ openclaw gateway

# 发送消息
pili4@DESKTOP:~$ openclaw message send --to +1234567890 --message "Hello"

# 与助手对话
pili4@DESKTOP:~$ openclaw agent --message "帮我写一首诗"

# 检查状态
pili4@DESKTOP:~$ openclaw doctor

# 更新OpenClaw
pili4@DESKTOP:~$ npm update -g openclaw@latest

七、常见问题

Q1: npm安装报错权限不足？

尝试使用pnpm代替npm，或者配置npm的prefix路径：

pili4@DESKTOP:~$ mkdir -p ~/.npm-global
pili4@DESKTOP:~$ npm config set prefix '~/.npm-global'
pili4@DESKTOP:~$ echo 'export PATH=~/.npm-global/bin:$PATH' >> ~/.bashrc

Q2: Node版本过低？

使用fnm管理Node版本：

pili4@DESKTOP:~$ fnm install 22 --force
pili4@DESKTOP:~$ fnm use 22

Q3: WSL2内存占用过高？

创建~/.wslconfig限制内存：

pili4@DESKTOP:~$ notepad.exe ~/.wslconfig 
[wsl2]
memory=4GB
processors=2

然后重启WSL：

PS C:\Windows\system32> wsl --shutdown

Q4: 如何使用Docker安装？（可选）

Docker是可选的，如果你想使用Docker方式：

pili4@DESKTOP:~$ git clone https://github.com/openclaw/openclaw.git
pili4@DESKTOP:~$ cd openclaw
pili4@DESKTOP:~/openclaw$ ./docker-setup.sh

🎉 恭喜完成！

现在你已经成功在Windows上安装了OpenClaw！快去访问控制台开始使用吧。

如果有任何问题，可以参考官方文档：https://docs.openclaw.ai
